# Light Control Web

Dự án mô phỏng điều khiển bật/tắt đèn bằng **HTML, CSS, JavaScript thuần**.

## Cấu trúc thư mục

```text
/light-control-web
│── index.html
│── style.css
│── script.js
│── README.md
```

## Cách chạy

1. Tải hoặc copy thư mục `light-control-web` về máy.
2. Mở file `index.html` bằng trình duyệt (Chrome, Edge, Firefox...).
3. Nhấn nút **BẬT ĐÈN** hoặc **TẮT ĐÈN** để thay đổi trạng thái đèn.

## Chức năng

- Tiêu đề trang: **Điều khiển đèn IoT**.
- Bóng đèn ở giữa màn hình.
- Hiển thị trạng thái: **Đèn đang TẮT / Đèn đang BẬT**.
- Khi bật đèn:
  - Bóng đèn chuyển vàng và phát sáng.
  - Nền giao diện sáng hơn.
  - Trạng thái đổi thành **Đèn đang BẬT**.
- Khi tắt đèn:
  - Bóng đèn chuyển xám.
  - Nền giao diện tối hơn.
  - Trạng thái đổi thành **Đèn đang TẮT**.

## Ghi chú

- Không dùng framework.
- Đã tách riêng HTML/CSS/JS theo đúng chức năng.
- Giao diện responsive cho điện thoại và laptop.
